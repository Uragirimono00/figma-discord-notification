# figma-discord-notification

### Tech Stacks:
- Java 17
- Spring Boot 3.1.3
- JUnit5 (Test)
- Lombok

### How To Use
> application.yml
```yml
figma:
  apiUrl: https://api.figma.com/v1
  apiToken: {yourFigmaApiToken}
  fileId: {yourFigmaFileId}
discord:
  webhookUrl: {yourDiscordWebHookUrl}
```

And run application
